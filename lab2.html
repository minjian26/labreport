Results & Analysis

First, I explore the codes to draw a random curve. The codes are attached below. Except drawing specific curves, then it starts the codes with "import random" to import random curves or shapes. Then, it initialize the background of the turtle canvas so that it won't go beyond the boundary. Next, it is to define a function "random walk" with steps and step length. The "jump" is to set the initial position with certain speed. The for loop is to repeat the steps with the forward and right direction. Last, it executes the function "random_walk" with certain steps and step_length. When I run the code, it shows different curves as it imports random curves at the beginning. 


import random
canvas_width = 1000
canvas_height = 800
initializeTurtle(initial_window_size=(canvas_width, canvas_height), mode='auto')

def random_walk(steps, step_length):
    speed(0)          
    penup()
    jump(450, 450)     
    pendown()

    for _ in range(steps):
        forward(step_length)              
        right(random.uniform(-180, 180))   # Turn by a random angle between -180 and 180 degrees

steps = 1000
step_length = 10

random_walk(steps, step_length)

show()
